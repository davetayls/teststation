Google Maps text-indent bug

This bug appears in iOS 5.1 where the size of a negative text-indent value is greater than -9999px

It causes the header content to disappear.

CSS


.container {
    position: relative;
    z-index: 100;
}
    .indent {
        display: block;
        text-indent:-999999px;
    }
    .map {
        background-color:Gray;
        height:213px;
        width:220px;
    }

This will disappear when we load the map in iOS 5.1
